# Nihongo Tools

![Nihongo Tools logo](logo.png)

Desktop-приложение на Kotlin для небольших утилит по изучению японского языка.

Приложение открывается в одном окне увеличенной высоты и использует системные диалоги выбора файлов и папок. Для папок запоминается последняя использованная директория.

Сейчас в приложении доступны:

- подсчет кандзи в документах с экспортом в CSV
- скачивание MP3 из фрагментов словаря Marugoto
- The Kanji Map Audio Thief для слов и MP3 со страниц `thekanjimap.com`

## Важно для macOS

При первом запуске macOS может показать предупреждение вида:

`Apple could not verify "Nihongo Tools" is free of malware`

Это происходит не потому, что приложение опасно, а потому, что оно пока не подписано и не нотарифицировано через Apple Developer. В планах этого пока нет, так как нужен платный аккаунт для сертификации.

Как открыть приложение:

1. Скопируйте приложение в `Applications` или в любую удобную папку.
2. Нажмите по приложению правой кнопкой мыши или `Ctrl + Click`.
3. Выберите `Open`.
4. В появившемся окне снова нажмите `Open`.

Если второй кнопки `Open` нет, то нужно нажать `Done`. Далее, можно сделать так:

1. Откройте `System Settings`.
2. Перейдите в `Privacy & Security`.
3. Прокрутите страницу вниз.
4. Найдите сообщение о заблокированном `Nihongo Tools`.
5. Нажмите `Open Anyway`.

После этого приложение обычно открывается как обычно и при следующих запусках больше не блокируется.

## Возможности

### Счетчик кандзи

- принимает документы разных форматов через Apache Tika
- умеет исключать кандзи из отдельного файла
- сохраняет результат в CSV
- поддерживает выбор подпапки для результата
- показывает прогресс обработки

Поддерживаются многие форматы, которые умеет читать Tika, включая `txt`, `pdf`, `docx`, `doc`, `epub`, `fb2` и другие.

### Загрузка аудио Marugoto

- принимает HTML-подобный текстовый фрагмент
- находит пары `_jpn` и `data-audio`
- скачивает `.mp3` с сайта Marugoto
- сохраняет файлы с японскими именами
- поддерживает выбор подпапки для результата
- показывает прогресс скачивания

### The Kanji Map Audio Thief

- принимает кандзи, например `食`
- загружает страницу `https://thekanjimap.com/<кандзи>`
- извлекает все слова со страницы, у которых доступно аудио
- показывает слова в таблице с чекбоксами
- позволяет отметить все или снять выделение со всех
- скачивает все отмеченные слова одной кнопкой
- сохраняет файл как имя слова без чтения в скобках, например `食堂.mp3`
- поддерживает выбор папки назначения заранее

## Как пользоваться

### Подсчет кандзи

1. Выберите исходный файл.
2. При необходимости выберите файл с исключениями.
3. При желании укажите имя подпапки.
4. Нажмите `Запустить подсчет`.
5. Выберите папку назначения.
6. Получите CSV с колонками `kanji,count`.

Если поле подпапки пустое, CSV сохраняется прямо в выбранную папку.

### Скачивание аудио Marugoto

1. Вставьте HTML-подобный фрагмент.
2. При желании укажите имя подпапки.
3. Нажмите `Скачать MP3`.
4. Выберите папку назначения.
5. Дождитесь завершения загрузки.

Если поле подпапки пустое, аудиофайлы сохраняются прямо в выбранную папку.

### The Kanji Map Audio Thief

1. Выберите папку для сохранения MP3.
2. Введите кандзи, например `食`.
3. Нажмите `Получить слова`.
4. Дождитесь появления таблицы слов с переводами.
5. Отметьте нужные строки или используйте `Отметить все` / `Снять все`.
6. Нажмите `Скачать отмеченные`.

Файл будет сохранен в выбранную папку под именем слова без чтения в скобках. Например, `食堂（しょくどう）` сохранится как `食堂.mp3`.
