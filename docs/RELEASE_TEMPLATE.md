## Nihongo Tools x.y.z

Desktop utilities for learning Japanese.

### Included in this release

- Kanji counter with CSV export
- Marugoto audio downloader

### Downloads

- macOS: `Nihongo Tools-x.y.z.dmg`
- Windows installer: `Nihongo Tools-x.y.z.msi`
- Windows executable installer: `Nihongo Tools-x.y.z.exe`

### Installation

- macOS: open the `DMG` and move the application to `Applications`
- Windows: run the `MSI` or `EXE` installer

### Notes

- macOS may show a warning on first launch if the app is unsigned
- if that happens, open `System Settings -> Privacy & Security` and allow the app manually
